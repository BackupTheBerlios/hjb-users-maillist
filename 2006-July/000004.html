When running quick demos and manual functional tests,  I tend to use the
python client for HJB, PyHjb <http://hjb.python-hosting.com>, and
ActiveMQ <http://www.activemq.org>.  What messaging product are you
using ?

I'm not sure precisely what info you are looking for for, but the
following list of URLs might help:

    Instructions for downloading PyHJB:
http://hjb.python-hosting.com/wiki/GettingStarted

    There are simple code samples for using PyHJB:
http://hjb.python-hosting.com/wiki/ExampleCode

    The PyHjb distribution comes with a number of functions that
demonstrate the use of python to communicate with a JMS server via HJB.
Information about these demos is available here:
http://hjb.python-hosting.com/wiki/WorkingDemos.

    The instructions for configuring several different messaging
providers  (ActiveMQ, SwiftMQ, JBoss Messaging and Websphere MQ) for use
with HJB are included in the PyHJB distribution, and they can be
accessed online via Trac:

    ActiveMQ:
http://hjb.python-hosting.com/file/trunk/samples/data/activemq/testing-a
ctivemq.rst?rev=44

    SwiftMQ:
http://hjb.python-hosting.com/browser/trunk/samples/data/swiftmq/testing
-swiftmq.rst?rev=latest

    WebSphereMQ:
http://hjb.python-hosting.com/browser/trunk/samples/data/wsmq/testing-ws
mq.rst?rev=latest

    JBossMessaging:
http://hjb.python-hosting.com/browser/trunk/samples/data/jboss/testing-j
boss.rst?rev=latest

My apologies if you've already looked at these, and want more!
